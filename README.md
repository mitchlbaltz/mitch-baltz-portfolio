# Mitch Baltz Portfolio

A static one-page professional portfolio for Mitch Baltz, designed for Democratic campaign communications, political research, public affairs, advocacy, digital strategy, and organizing opportunities.

## Technology

- Plain HTML in `index.html`
- CSS in `assets/styles.css`
- Minimal vanilla JavaScript in `assets/script.js` for the mobile navigation
- No build step, package manager, server-side code, or third-party libraries

## File structure

```text
.
├── index.html
├── 404.html
├── .nojekyll
├── README.md
└── assets/
    ├── favicon.svg
    ├── mitch-baltz-resume.pdf
    ├── script.js
    └── styles.css
```

## Publishing on GitHub Pages

1. Push this branch to GitHub.
2. Open the repository settings and go to **Pages**.
3. Select the branch and root folder as the Pages source.
4. Publish the site. All asset paths are relative, so the site works from a project URL such as `/mitch-baltz-portfolio/`.

## Resume

The source resume PDF is stored at `assets/mitch-baltz-resume.pdf` and linked with download buttons in the navigation, hero, and contact sections.
